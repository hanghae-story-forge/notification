# 유비쿼터스 언어 업데이터 (Ubiquitous Language Updater)

도메인 변경을 감지하고 유비쿼터스 언어 문서를 자동으로 업데이트하는 에이전트입니다.

## 사용법

이 스킬은 다음과 같은 상황에서 자동으로 활성화됩니다:

- 도메인 계층 코드 변경 후 문서 업데이트 요청
- "유비쿼터스 언어 업데이트해줘"
- "도메인 변경사항을 문서에 반영해줘"
- "ubiquitous language update"

## 작업 흐름

### 1. 변경 감지

git diff를 사용하여 최근 도메인 계층 변경 사항을 확인합니다.

### 2. 도메인 분석

변경된 파일들을 읽어 다음 정보를 추출합니다:

- **새로운 도메인**: 새로 추가된 애그리거트 루트
- **값 객체**: 새로 추가되거나 변경된 VO
- **도메인 이벤트**: 새로운 이벤트 타입
- **비즈니스 로직**: 새로운 메서드와 그 의미
- **관계 변경**: 애그리거트 간 관계 변화

### 3. 문서 업데이트

추출된 정보를 바탕으로 `.claude/docs/ubiquitous-language/ubiquitous-language.md`를 업데이트합니다:

- 도메인 모델 섹션에 새로운 도메인 추가
- 값 객체 테이블 업데이트
- 도메인 이벤트 목록 갱신
- 용어 사전에 새로운 용어 추가
- 관계 다이어그램(필요시) 업데이트

## 주의사항

1. 도메인 변경이 없을 경우 사용자에게 알리고 종료합니다
2. 문서의 기존 구조를 유지하면서 변경 사항만 반영합니다
3. 용어의 한글/영문 매핑은 일관성을 유지해야 합니다
4. 기존 용어를 삭제하지 않고, 새로운 용어를 추가합니다
